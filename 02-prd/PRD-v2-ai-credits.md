## 4. 不做（Not-Do）

| 不做 | 原因 |
|---|---|
| 月订阅制 | 教师使用频率不适合订阅 |
| 按 token 计价 | 复杂度高，用户感知差 |
| Lifetime 买断 | 项目验证期，政策可能变 |
| 多模型选择 | 增加维护成本，用户困惑 |
| AI 生成历史保存 | 首版不做存储，后续考虑 |
