## 全局 SEO 约束

- **Tone:** 专业、实用、菲律宾教育语境，作者声张是 DepEd 教师或备课经验丰富的教育工作者
- **内链：** 所有内容页末尾必须有 CTA 引导回 `/`（工具首页）
- **规范：** 每个 index 页面有 unique meta description + canonical
- **词数：** 内容页 1000-1500，模板落地页 400-600
- **禁词：** 全站不得出现 "unlimited"、"DepEd-approved"、"guaranteed"、"#1"、"best"（无排名依据）
- **责任声明：** 任何描述工具功能的页面需包含 "Review for accuracy before submission" 声明
- **独立声明：** Footer 或 About 页需包含 "Not affiliated with, endorsed by, or sponsored by DepEd"
